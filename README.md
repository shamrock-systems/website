# Shamrock Website

Landing page for **Shamrock** — a Discord-to-Roblox management integration platform.

## Pages

- **Home** (`index.html`) — Hero, overview, feature cards, tech stack
- **Features** (`features.html`) — Detailed breakdown of each capability
- **Roadmap** (`roadmap.html`) — Development timeline and milestones
- **About** (`about.html`) — Project status, license, contact

## Built With

- Vanilla HTML / CSS / JS
- [Inter](https://fonts.google.com/specimen/Inter) font family

## Local Dev

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.
