Roadmap

The path to public release and beyond.

Phase 1

Core Integration & Verification

OAuth2-based Roblox verification, Discord role mapping, and secure account linking infrastructure.

Phase 2

Administration Dashboard

A web-based dashboard for managing permissions, viewing audit logs, and configuring integration settings without touching commands.

Phase 3

Public API & Documentation

Expand public API endpoints with comprehensive documentation, enabling community-built extensions and integrations.

Phase 4

Beta Release

Open beta for wider community testing. This release will include all core features, the dashboard, and documented API.

Future

Stable Release & Beyond

Battle-tested stable release with ongoing improvements based on community feedback, expanded platform support, and advanced analytics.
