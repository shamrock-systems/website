📋 Project Status

Shamrock is currently in active development. Access is limited to select servers as we refine core features and ensure stability before a wider release.


We are building Shamrock to be the go-to solution for Discord-Roblox integration, focusing on security, performance, and ease of use.

🔧 Built With

  • Discord.js — Discord bot interactions
  • Roblox API — Roblox user/group data
  • Node.js — Backend runtime
  • Cryptographic libraries for secure linking
  • Asynchronous job queue for reliable sync
